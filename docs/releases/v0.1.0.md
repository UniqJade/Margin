# Margin v0.1.0

Margin v0.1.0 is the first source-only personal release of an English →
Simplified Chinese reading companion for Apple Books.

## Highlights

- Select text in Apple Books on Mac, press `⌃⌥M`, and read the result without
  switching to a general-purpose translation app.
- Translate words with compact dictionary-style structure or passages as
  natural Chinese prose.
- Inspect sentence-level semantic alignment without generating a second,
  conflicting translation.
- Keep credentials in the device-only Keychain and retain only a bounded local
  response cache plus items the user explicitly saves.
- Build from auditable Swift/SwiftUI source under the MIT License.

## Verified scope

- Primary path: macOS 26.5, Apple Books 8.5.
- Deployment targets: macOS 15 and iOS/iPadOS 18.
- Certified provider configuration: `deepseek-v4-flash`.
- iPhone/iPad shells compile, but Apple Books selection delivery remains
  experimental until physical-device testing is recorded.

The pre-locked, single-evaluator 40-passage blind comparison met all three
v0.1.0 gates: DeepSeek was preferred for naturalness in 37/40 passages, was
equal or better for accuracy in 37/40, and had no marked major semantic errors.
This result is bounded evidence, not a claim of universal superiority. See
[`docs/evaluation.md`](../evaluation.md) for the method and limitations.

## Install

This release intentionally contains no `.app`, DMG, or other binary asset.
GitHub's automatically generated source archives are the release downloads.

Build and sign your personal copy by following
[`docs/building.md`](../building.md). You need full Xcode, XcodeGen, an Apple
Development identity with its private key, and your own DeepSeek API key.

## Important limitations

- English → Simplified Chinese only.
- Network and cloud provider required; no offline dictionary or local model.
- Direct client-side BYOK is appropriate only for a locally built personal
  prototype.
- AI-generated definitions and translations can be wrong and do not cite a
  licensed authoritative dictionary.
- Semantic alignment is sentence-level. A single long grammatical sentence can
  form one block and therefore look nearly identical to Natural Translation.
- The verified Mac shortcut requires user-approved Accessibility access.
- Mobile Apple Books handoff has not yet been verified on a physical device.

## Privacy reminder

Do not paste API keys, private book excerpts, provider response bodies, signing
files, or populated `Local.xcconfig` values into public issues. See
[`SECURITY.md`](../../SECURITY.md) before reporting a security problem.
